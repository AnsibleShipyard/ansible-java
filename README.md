ansible-java
=============

[![Build Status](https://travis-ci.org/AnsibleShipyard/ansible-java.svg?branch=master)](https://travis-ci.org/AnsibleShipyard/ansible-java)

Java Playbook for Ansible

This playbook will install Java OpenJDK.


## Docker Usage

For more information see: [ansible-java Docker page](https://registry.hub.docker.com/u/ansibleshipyard/ansible-java/)

To start using this docker:

```shell
docker pull ansibleshipyard/ansible-java
```
